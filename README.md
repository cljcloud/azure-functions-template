# azure-functions lein template

A Leiningen template for serverless ClojureScript app on Azure Functions.

## Usage

Run:

    # create a new project using template
    $ lein new com.cljcloud/azure-functions my-app

    # go to app folders
    $ cd my-app

    # install deps
    $ lein deps
